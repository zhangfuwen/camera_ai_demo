# 安装

1. 安装python https://www.python.org/downloads/
2. 执行以下命令

Under windows:
```python
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python food_detection_server.py

```

Under Linux/Mac OS:

```python
python3 -m venv .venv
source ./.venv/bin/activate
pip install -r requirements.txt
python3 food_detection_server.py

```
